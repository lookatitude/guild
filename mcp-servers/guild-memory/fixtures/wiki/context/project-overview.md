# Project overview

The Guild plugin is a team-of-specialists system for Claude Code. It orchestrates
brainstorm, plan, execute, review, verify, and reflect phases across thirteen
domain specialists spanning engineering, content, and commercial groups.

The architecture emphasizes a reliable spine first and defers self-evolution
until real telemetry data exists. BM25 wiki search is an optional scale
transition that activates above two hundred pages.
