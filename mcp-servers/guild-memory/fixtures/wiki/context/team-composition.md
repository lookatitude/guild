# Team composition rules

Teams cap at six specialists per task. The recommended default range is three
to four. Lifting the cap requires an explicit allow-larger flag. Team
composition prefers specialists whose description-similarity to the spec
exceeds the boundary threshold while avoiding adjacent overlaps flagged by
boundary scan.
