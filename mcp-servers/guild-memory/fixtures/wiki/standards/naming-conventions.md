# Naming conventions

Specialists use kebab-case role names (backend-engineer, technical-writer).
Skills use kebab-case directory names under skills/. Commands use the guild
namespace prefix. Wiki pages use lowercase hyphenated filenames ending in
markdown. Run identifiers use ISO date prefixes followed by a short hex slug.
