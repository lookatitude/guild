## Contract walk-through

Read the 4 P2 skills and commit `8628dfe` (P2 group-review fix pass). The chain after the fix is coherent end-to-end:

1. **`guild:wiki-ingest`** (`skills/knowledge/wiki-ingest/SKILL.md`) consumes a source (URL, file, pasted doc), writes the raw artifact under `.guild/raw/sources/<slug>/`, then writes a wiki page under `.guild/wiki/<category>/<slug>.md` with the full ┬¦10.1.1 base frontmatter. 2. **`guild:wiki-query`** (`skills/knowledge/wiki-query/SKILL.md`) searches the wiki. Pre-fix the filter accepted plural dir names, which never matched the singular frontmatter; the fix in `8628dfe` (change I3) resolved that drift.

3. **`guild:wiki-lint`** (`skills/knowledge/wiki-lint/SKILL.md`) validates every page under `.guild/wiki/<category>/` against ┬¦10.1.1 base (Check #1), and pages under `.guild/wiki/decisions/` additionally against ┬¦10.3 ADR-lite additions (Check #7).
